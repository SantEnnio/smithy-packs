# CS1061 — Member missing on type

Meaning: the type exists but the named member does not.

Common Unity causes:

- API renamed between versions (e.g. `Rigidbody2D.velocity` →
  `Rigidbody2D.linearVelocity` in Unity 6).
- Misspelled member or wrong type.
- Extension method's `using` is missing.

Repair recipe:

1. Read the line and the surrounding usage.
2. Check the type and look up the correct member.
3. Apply a minimal `patch`.
